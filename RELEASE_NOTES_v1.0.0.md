# StarTrack Automation — Release Notes
## Version 1.0.0 | March 2026

---

## 🎉 Initial Release

We are excited to announce the first official release of **StarTrack Automation** — a smart expense tracking and management app built for teams and enterprises.

---

## ✨ What's New

### 🔐 Google Sign-In
- Secure one-tap authentication using your Google account
- Automatic user profile creation on first login
- Session persistence across app restarts

### ➕ Add Expense
- Create detailed expense entries with multiple line items
- Attach expense type, amount, and description per entry
- Select approver and expense date
- Automatic total amount calculation
- Expenses saved instantly to the cloud

### 📄 My Expenses
- View all your submitted expenses in one place
- Filter expenses by status — All, Pending, Approved, Rejected
- Color-coded status badges for quick visibility
- Summary card showing total amount and expense count
- Pull to refresh for latest data

### 👤 Profile Screen
- View and edit your profile details
- Change profile photo from your gallery
- Update display name and phone number
- View account activity — last login and account creation date
- Secure logout with confirmation
